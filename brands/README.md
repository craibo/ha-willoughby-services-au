This directory is reserved for branding assets used by HACS and Home Assistant.

Add an `icon.png` (and optionally `logo.png` or `background.png`) under a subdirectory
named after the integration domain, for example:

- `brands/willoughby_services/icon.png`

For this project, the icon should be inspired by Willoughby City Council colours and
theme, but must not use official logos or protected marks.
