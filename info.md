## Willoughby Waste and Street Sweeping Services

This Home Assistant integration provides date sensors for properties within the Willoughby City Council area (NSW, Australia):

- Next red (general waste) bin collection
- Next yellow (recycling) bin collection
- Next green (garden) bin collection
- Next scheduled street sweep
- Next autumn, winter, and summer bulky waste collections

Configuration is done via the Home Assistant UI by providing a Willoughby street address and/or a council geolocation ID. The integration queries the official council waste services endpoint on a daily schedule.
